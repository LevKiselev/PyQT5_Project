# Проект по PyQT ГОЛОВОЛОМКИ

Программа написана в рамках учебного курса Яндекс Лицей и объединяет три головоломки:
1. Угадай число
2. Открой картинку
3. Ход конем
Главное окно содержит четыре закладки. На первой (стартовой) закладке пользователь может
ввести свое имя для сохранения в базе данных результатов игр, а также выбрать игру.
В каждой игре ведется подсчет ходов и времени. Для каждой игры предусмотрены возможности
сброса, просмотра правил и статистики (с сортировкой по столбцам).


## Содержимое архива


### Документация

readme.md - этот файл
requirements.txt - список используемых внешних модулей, созданный с помощью pip freeze


### Файлы, необходимые для запуска и работы программы

main.py - главная программа
numbers_game.py - модуль с описанием класса игры "Угадай число"
picture_game.py - модуль с описанием класcов игр "Откой картинку" и "Ход конем"
rules.py - модуль с описанием класcов информационных окон

Модули графического интерфейса, преобразованные из ui-файлов с помощью PyUIC:

main_ui.py - главное окно программы
numbers_ui.py - виджет игры "Угадай число"
picture_ui.py - виджет игр "Открой картинку" и "Ход конем"
rules_ui.py - окно правил игры
info_ui.py - информационное окно
start_ui.py - виджет стартовой вкладки
userlist_ui.py - окно списка игроков и статистики

Модули ресурсов, преобразованные из qrc-файлов и изображений с помощью Pyrcc5

start_rc.py - иконки кнопок стартовой вкладки (картинки стащил из сети)
pics_rc.py - фоновые изображения для игр (фотки мои)

records.db - база данных со списком игроков и статистикой игр.
В случае ее отсутствия программа создает новый пустой файл.


### Запускаемый файл

В каталоге dist лежит созданный с помощью pyinstaller запускаемый файл **puzzles.exe**
Проверена работа в Windows 8/10


### Дополнительные файлы

В каталоге src лежат ui- и qrc-Файлы для QtDesigner, а также каталоги файлов с изображениями.
Для работы программы они не нужны, т.к. уже преобразованы в py-модули.


## Сборка и запуск программы

Для работы программы необходимо отдельно установить только модуль PyQT5.
Остальные модули, используемые в программе, являются встроенными и установки не требуют:
sys, sqlite3, datetime, random.

Запускать нужно main.py, при этом остальные 12 вышеперечисленных py-файлов должны лежать с ней
в одном каталоге. В теории должно работать... Во всяком случае, у меня работает :)
